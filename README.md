# Rust Nirvana

A warm, rust-toned dark theme for [Zed Editor](https://zed.dev), crafted for Rust developers.

Deep browns, burnt oranges, sage greens — the color of oxidized iron and focused flow states.

## Features

- 🦀 Optimized for **Rust** syntax highlighting
- 🎨 Colorized brackets — `()` blue · `{}` pink
- 🌿 Calm green strings, no eye fatigue
- 🔥 Warm orange/red type system emphasis
- 🌑 Deep dark background `#1a1818`

## Installation

Search **Rust Nirvana** in Zed Extensions (`cmd+shift+x`).

## Recommended Settings

Add to your `settings.json` for the full experience:

```json
"colorize_brackets": true
```

This enables bracket colorization — `()` render in blue, `{}` in pink, making nested Rust code dramatically easier to read.
